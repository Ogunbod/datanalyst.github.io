Dental Care Analysis

This project showcased an in-depth analysis and visualization of international Dental Care data using Excel. Advanced techniques were utilized, including VLOOKUPs for data consolidation, INDEX & MATCH functions for enriched data retrieval, and IF functions for categorization, significantly enhancing dataset clarity and usability. Careful data formatting—such as date adjustments, size specifications, and currency annotations—along with diligent duplication checks, laid the groundwork for a meticulously structured dataset. The outcome was a comprehensive Excel dashboard comprising pivot table, line chart, bar charts, timeline, and three slicers. This dashboard offers an interactive platform for exploring Dental Care trends, highlighting the project's contribution to simplifying complex data analysis and enabling insightful decision-making.
